## 1. 项目概述

### 1.1 项目名称
云上田园 · 河南云农场

### 1.2 使命
**让每个城市家庭都有一块"看得见的安心农田"。**

### 1.3 核心模式

1. **认养制**：在河南（建议周口/驻马店）租赁土地 50 亩，切分为 10–20 ㎡ 标准模块，通过小程序与 Web Portal 出售「年度认养权」。
2. **可参与**：用户可在小程序内选地、选作物、发送"浇水/施肥/除草/拍张照"指令（由场内农技员或后端摄像头自动执行）。
3. **可观看（v3 澄清）**：每个认养田块部署太阳能 4G 摄像头，**用户在自己的小程序/Web Portal 端实时查看自己的地块画面**（私有远程监控，1 用户 → 1 摄像头）。摄像头并非公开直播，没有第三方观众。进阶版/亲子版套餐还可远程控制云台。
4. **可收获**：作物成熟后按约定批次冷链寄送到用户手上，每户保底 10–20 斤。
5. **可分享**：用户把生长报告、收获照片分享到社交平台，形成二次传播。
6. **可销售（v3 澄清）**：自营地（未被认养的 20–30 亩）同步种植热门作物（水果玉米、贝贝南瓜、蜜薯等），主销售渠道是我们小程序/Web 内置的农产品商城，**抖音/视频号直播带货作为引流补充**（在抖音平台上直播带货，引流到我们小程序下单）。

### 1.4 三年战略分期
- **第一年（2026）**：单基地模式跑通，建立 C 端（小程序 + Web Portal）+ B 端 Web 后台 + 后端 API，完成首个 50 亩基地闭环。
- **第二年（2027）**：扩展 SKU 与增值服务（亲子研学、农场游学、企业团建），**规模化运营农产品商城 + 拓展抖音/视频号直播带货渠道**。
- **第三年（2028）**：复制一个华南基地，上线"多基地"版本，输出 SaaS 赋能给其他小型农场主。

## 2. 市场分析

### 2.1 用户痛点（需求侧）

| 痛点 | 表现 | 我们的回应 |
|---|---|---|
| 食品安全焦虑 | 农药残留、来源不明 | 全程**摄像头可见** + 种植日志，可追溯 |
| 亲子教育刚需 | 城市儿童缺乏自然课 | 亲子版套餐 + 线上自然课 + 孩子名字立牌 |
| 陪伴式情感消费 | 云吸猫、云旅游流行 | **24h 摄像头监控自己的专属田地**，带"主角感" |
| CSA 参与感弱 | 传统配送单向、无互动 | 指令交互 + 生长报告 + 命名权 |
| 企业福利同质化 | 月饼/大米/卡券千篇一律 | 企业田 + 员工远程观看摄像头 + 定制包装 |

**v3 变更**：删除原"全程直播 + 种植日志"提法，改为"全程摄像头可见 + 种植日志"，避免"直播"这种公开广播语义混淆。

### 2.2 市场规模测算（TAM / SAM / SOM）
- **TAM**：一线城市常住人口约 8000 万，家庭数约 2600 万。
- **SAM**：对"食品安全+自然教育"付费意愿较强的中高收入家庭，保守估 260 万（占比 10%）。
- **SOM（可获得市场）**：首年渗透 0.03% ≈ 800 户；三年复合渗透 0.2% ≈ 5000 户。
- **客单价区间**：499 – 999 元/年（认养），100 – 300 元/单（农产品商城），3000 – 20000 元/年（企业端）。

### 2.3 目标用户画像

**核心人群 A – "自然教育妈妈"**
- 年龄：30–42；职住：一线城市；年收入：家庭 40 万+
- 典型行为：知乎/小红书"有机""宝宝辅食"重度读者；给孩子报自然课、博物馆亲子营
- 付费意愿：亲子版 699 元/年，次年续费靠"孩子认出自己的田 + 摄像头里看到果实成熟"

**核心人群 B – "松弛感白领"** ★ v3 加强
- 年龄：25–35；单身或丁克；消费力强
- 典型行为：买多肉、养猫、订花、追综艺
- 付费意愿：基础版 499 元/年，看中"情绪价值 + 社交货币"
- **场景关键**：上班累的时候在公司电脑大屏上打开 Web Portal，看自己田里的番茄长势 5 分钟，治愈班味

**次核心人群 C – "企业行政/HR"**
- 诉求：低单价高触达的员工福利、ESG 故事点
- 付费意愿：单公司 5000–20000 元/年

## 3. 产品与服务设计

### 3.1 认养套餐体系（核心产品）

| 版本 | 面积 | 年费 | 核心权益 |
|---|---|---|---|
| **基础版** | 10 ㎡ | 499 元 | 1 种作物，保底收获 10 斤，包邮 2 次，**共享摄像头轮播查看**，每月生长报告 |
| **进阶版** | 15 ㎡ | 799 元 | 2 种作物，保底 20 斤，包邮 3 次，**专属可遥控摄像头（PTZ 云台）**，优先场景选择权 |
| **亲子版** | 10 ㎡ | 699 元 | 基础权益 + 3 节线上自然教育课 + 实体种植工具包 + 孩子名字立牌 + 生长纪念相册 |
| **企业版** | 50 ㎡ 起 | 5000 元起/年 | 企业 Logo 立牌、员工分账号（最多 50 人共享地块查看），定制包装、年度 ESG 报告 |

**作物策略**：
- 优先：耐储运 + 高观赏性 —— 红薯、胡萝卜、土豆、南瓜、草莓、小番茄、香椿、大蒜
- 避开：易腐叶菜（仅秋冬限量）
- 深加工：萝卜干、豆角干、辣椒酱、芝麻盐（解决物流与损耗）

### 3.2 互动玩法
- **指令系统**：浇水、施肥、除草、捉虫、拍照特写（有冷却时间，每周限额，防刷屏）
- **拍照特写（v3 升级）**：用户点"拍张照"后，**后端自动从摄像头抓帧**，2 秒内画面闪一下并写入生长日记。无需农技员人工操作（非农技员手动拍照后上传的备用通道）
- **成长里程碑**：发芽、开花、结果、收获 4 节点自动推送，附短视频
- **命名权**：给田块起名 / 给作物起名（亲子版可印在立牌上）
- **土地流转**：用户可把地块转赠给朋友，促裂变
- **年度回忆录**：年底自动生成本季生长时间轴 + 短视频，方便用户转发

### 3.3 农产品商城 + 抖音直播带货（辅助产品）★ v3 重写

**自营地 20–30 亩**：种水果玉米、贝贝南瓜、蜜薯等

**销售渠道**：
1. **主渠道：app 内商城**（小程序 + Web Portal）
   - 9.9 元尝鲜装（引流）
   - 99 元月卡箱
   - 999 元年卡（12 次配送）
2. **辅助引流：抖音/视频号外部直播带货**
   - 每天 4–6 小时，固定栏目
     - 周一"今日田间巡视"
     - 周四"挖红薯大赛"
     - 周六"摘草莓直播"
   - 在抖音/视频号上播，挂小黄车直接卖货 / 引流到我们小程序
   - **这部分直播运营走抖音生态,不在我们 app 内做推流分发**

**v3 关键澄清**：v2 项目书把"自营直播电商"作为一个独立的产品线表述，容易让人误以为我们要在自己 app 内做直播 tab。v3 明确：自营销售就是"农产品商城",抖音直播只是营销渠道,与"用户私有摄像头看农场"是两个完全不同的东西。

### 3.4 增值服务（长尾）
- 生长报告纸质相册（+88 元/季）
- 特殊种子/稀缺品种（+50–200 元）
- 农场实地研学（周末 1 日游，含车费 580 元/人）
- 婚礼/生日田（纪念日种植服务，1500–3000 元）

### 4.2 技术选型（与软件架构 v2 §9 一致）

| 层 | MVP 选型 | 理由 |
|---|---|---|
| 微信小程序 | Taro 4 + Vue 3 | 现状,保留 |
| C 端 Web Portal | **Vue 3 + Vite + vue-router(history)** | 独立项目,SEO 友好,大屏看摄像头清晰 |
| Web 后台 | Vue 3 + Element Plus + Vite | 成熟生态,招聘容易 |
| 后端 | **NestJS** (Node 20) | 团队 JS 栈统一,模块化清晰 |
| ORM | **Prisma** | TS 类型推导优,DX 好 |
| 数据库 | MySQL 8 | 业务数据 |
| 缓存 | Redis 7（可选） | MVP 阶段无 Redis 也能跑 |
| 对象存储 | 阿里云 OSS / Cloudflare R2 | 头像 / 抓拍照片 / 商品图 |
| **摄像头** | **萤石云 OpenAPI** | **海康设备原生,SaaS 方案,零运维** |
| 支付 | 微信支付（主）+ 支付宝（次） | |
| 物流 API | 顺丰 / 京东物流 | |
| Monorepo | pnpm workspace | 4 apps + 3 packages |
| 部署 | Docker Compose 单机起步 | 月成本 < 200 元 |

**v2 → v3 关键变更**：
- ❌ 删除 v2 的"腾讯云 LVB + 海康 ISUP"自建流媒体方案
- ✅ 选定**萤石云 OpenAPI** 作为唯一摄像头方案（SaaS,海康设备扫码即用）
- ❌ 删除 v2 的 RabbitMQ（MVP 用 Redis 队列即可）
- ✅ 后端定型 NestJS + Prisma（v2 是"NestJS 或 Spring Boot 二选一"）
- ✅ ORM 定 Prisma（v2 没指定）

### 4.3 关键技术点

1. **摄像头接入(私有监控,非直播)** ★
   - 海康/萤石设备扫码绑定到萤石云 → 后端调萤石 OpenAPI 取临时播放地址 → 小程序 `<live-player>` / Web `flv.js` 播放
   - 鉴权在 NestJS API 层做,**用户只能看自己认养地块的摄像头**,任何其他用户拿不到 url
   - 不存视频文件,需要回看走萤石云 7 天 SaaS

2. **指令执行闭环**
   - 用户点"浇水/施肥/除草" → API 创建 Command(pending) → 推 Web 后台 → 农技员手机端确认 → 上传完成照片 → API 写 JournalEntry → 用户订阅消息推送
   - 用户点"拍张照" → API 自动调萤石抓帧 → 存 OSS → 自动写 JournalEntry,用户无感知 2 秒完成

3. **消息推送**
   - 微信订阅消息（服务通知、生长节点、收获预告、摄像头离线告警）
   - Web Portal 用户走短信 + 浏览器 Web Push（可选）

4. **冷链物流**
   - 对接顺丰冷运 / 京东物流 API,订单号回传用户

5. **数据看板**
   - MVP 阶段直接 SQL 查 MySQL,等数据量上来再上开源 Metabase

### 4.4 部署架构(MVP 期)

- 阿里云 ECS / 腾讯轻量云 2c4g + Docker Compose
- C 端 web + 后台静态托管 Cloudflare Pages（自动 HTTPS,免费）
- 月成本：ECS ~100 元 + OSS ~30 元 + 域名 / SSL ~0,**总计 < 200 元/月**
